# Lyno Associate Website

Final responsive Lyno Associate website package.

## Run locally

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:5173
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Open Vercel and import the GitHub repository.
3. Use these settings:
   - Framework Preset: Vite
   - Install Command: npm install
   - Build Command: npm run build
   - Output Directory: dist
4. Click Deploy.

## Replace images

Images are in:

```bash
public/site-images
```

Replace an image using the same filename and refresh the site.
